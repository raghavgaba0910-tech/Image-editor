📸 Image Editor:
    A simple and interactive web-based image editor that allows users to upload, edit, and enhance images directly in the browser. 
    This project demonstrates core concepts of image manipulation using frontend technologies.

🚀 Features:
    1 Upload and preview images
    2 Apply filters (brightness, contrast, grayscale, etc.)
    3 Rotate and flip images
    4 Crop and resize functionality
    5 Basic drawing/editing tools
    6 Undo / Redo functionality
    7 Download edited images


🛠️ Tech Stack
    1 python
    2 OpenCV
    3 fundamentals
